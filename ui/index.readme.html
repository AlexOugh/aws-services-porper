For GitHub Authentication

Create OAuth Application

  • Follow the direction here to create an OAuth application 
    Use this site url for Authorization callback URL
  • Once completed, you will get Clinet ID and Client Secret that will be used in the next step.

Authenticate with a valid identity

  • Connect to Authentication url using a web browser 
    https://github.com/login/oauth/authorize?scope=user&client_id=<returned_client_id>&redirect_uri=<callback_url>&state=<any_unique_value>
  • You should be redirected to the 'callback_url' with 'code' & 'state' values 
    <callback_url>/?code=<code_value_after_successful_login>&state=<prev_state_value>

Inteface with Porper

  • Call Porper API interface with above 'code' & 'state' values after invited

Authenticate with a valid identity

      
    
        
  • Connect to Authentication url using a web browser
          
    https://accounts.google.com/o/oauth2/v2/auth?client_id=<returned_client_id>&response_type=code&scope=openid%20profile&redirect_uri=<redirect_uri>
    
        
    • 
        
  • You should be redirected to the 'redirect_uri' with 'code' value
          
    <callback_url>/?code=<code_value_after_successful_login>&authuser=0&session_state=<state>&prompt=consent#
